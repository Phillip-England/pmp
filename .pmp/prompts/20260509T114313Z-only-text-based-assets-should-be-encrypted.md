# only text based assets should be encrypted

imagine if we upload an image as an asset or an audio file as an asset maybe those sorts of files shouldn t be encrypted maybe only text base assets should be encrypted this allows us to have flexibility we can associate images with our project we can associate audio files with our project and that works out just fine but if we associate a PDF with our project or if we associate a text document with our project or some other sort of asset like a CVS file Etc we should be able to associate those assets with our project but those sorts of assets should be encrypted but things like images and simple things like that they should not be encrypted
