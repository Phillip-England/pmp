# New Prompts Via Audio

Imagine if we could enter prompts via audio? Here is what I think we should have. On the "new" page where we can add a new prompt, imagine if we have a button for a microphone. This mic can be clicked, and then when the user speaks their words will appear on the screen. controls to pause the mic or clear the text from the screen will be available as well. Here is how it will work: the users words will be recorded and will be scanned for the word 'dash'. For example lets imagine we have this text: "Things I like About You dash You are the best person ever". Well, "Things I like About You" will be the title and "You are the best person ever" will be the body text.

This allows a very easy way for users to describe their prompts using audio in the web interface.
