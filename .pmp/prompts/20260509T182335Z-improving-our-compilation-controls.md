# Improving our compilation controls

Right now, our compilation controls are kind of wonky. Here are the changes that I would like to have made. I would like the ability to compile all of my prompts in a single button. Then I would like the ability to compile all the prompts from the mark and a single button. I would like the ability to say whether or not this certain compilation process is going to impact the mark or not, I would like the ability to compile prompts based on a certain range. Maybe I compile the first prompt to the 40th prompt or whatever essentially I want to remove this need to select prompts for compilation and instead just have controls to compile directly, so I can compile from the mark I can compile all I can compile from a specific range. I can indicate whether I want that to impact the marked prompt or not, and all of that should be very easy to do in a little tiny control pa on the prompts page
