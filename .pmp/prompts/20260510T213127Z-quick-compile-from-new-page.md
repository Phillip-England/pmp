# Quick Compile From "New" Page

Imagine this, we might want to "quick compile" from the "New" page. Quick compile simply means running 'pmp compile --from-mark' so we just run that under the hood when the quick compile button is clicked. The quick compile button can be on the opposite side of the "save prompt" button associated with the form for making a new prompt. So right now to save a prompt i click the button on the left side with "Save Prompt" written on it, well to quick compile ill click the "Quick Compile" button which is on the right side, not the left. But in the same row, directly under the form.
