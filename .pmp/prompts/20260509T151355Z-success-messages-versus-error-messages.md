# Success Messages Versus Error Messages

Right now after filling out most forms on the site, I get indications, but they are not color coded based on success or failure. A good example is the password form to set a new password. When I go to do so, if the passwords do not match, my error message is not red so it can sort of get missed. Same thing for my success messages please and thank you.
