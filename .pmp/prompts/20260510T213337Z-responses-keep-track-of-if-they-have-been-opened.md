# Responses Keep Track of if They Have Been Opened

If a response has not been opened in the web browser, it is marked as unread. Unread responses are a different color in the UI. The amount of unread responses is in a tiny circle near the "Responses" nav button indicating to the user that some info has not been processes yet.
