# A page for an integrated terminal

I really like the idea of having an integrated terminal in this application. Here is kind of high envisioning. I have a page called the terminal page that I can essentially open up and it has a terminal active and these terminal sessions stay running irrespective of what happens elsewhere in the application so I can always come back to these terminal windows and there can be multiple windows active. They'll stay running, no matter what else happens in the application that would be ideal.
