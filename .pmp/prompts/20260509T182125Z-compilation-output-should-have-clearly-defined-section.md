# Compilation output should have clearly defined section

My compilation output should have three clearly defined sections. The first section is the instructions sections. The second section is the skills sections. The third sections is the prompt sections. Each of these sections have their own purpose in their own intent, and this should be laid out very clearly in the compilation output.
