# Local pmp Password History

We need some sort of way to see the password history for pmp on a certain machine. Here is why. Lets imagine you encrypt some assets, bind them into a document by making references to them, and then push them to github. Well, now those assets are essentially 'locked' in time until we pull them down and decrypt them for viewing. Well, imagine a situation where we forget the password which was used to encrypt the assets? Well, being able to view our on-disk password history might give us some clues. In the web UI simply give us a place to change our encryption password in the settings section. Then maybe just a popopen to see our password histroy.
