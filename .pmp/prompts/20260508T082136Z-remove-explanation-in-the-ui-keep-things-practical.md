# Remove explanation in the UI keep things practical

Right now, the UI has a whole bunch of explanatory text. We don't really need that. Keep the text to bare minimum and make the project more practical. Also make the font smaller in the project so that you can stash more stuff on a single page. We still want space between items, but this project is more about being able to see all of this stuff on a birds, iview and less about the UI being nice big and classy.
