# Do Not Scan Entire System For Projects

Right now we scan the entire system for projects which is a lot. Instead, maybe we set designated dirs on the system that are suitable for stashing projects within and then we scan those which will drastially cut down on the amount of scanning we have to do and thus decrease the load time.
