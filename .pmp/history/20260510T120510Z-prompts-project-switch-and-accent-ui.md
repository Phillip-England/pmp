# Prompts Project Switch And Accent UI

Prompts cards now open directly on click with hover and keyboard affordance, matching the no-view-button pattern used on responses. Switching projects now redirects to `/new` so the active project change is immediately visible, and the top working-directory banner is illuminated with a deliberate secondary accent used for contextual surfaces instead of primary actions.
