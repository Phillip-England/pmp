# Response hover affordance

Added a response-only clickable card treatment in the web UI: response cards now show a pointer cursor, lift slightly on hover, and highlight their border so it is obvious they can be opened. The change is scoped to the responses page rather than all prompt cards, which avoids changing prompt-page behavior unintentionally.
