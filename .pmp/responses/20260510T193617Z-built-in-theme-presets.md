# Built-in Theme Presets

Added a built-in set of eight site theme presets as primary/secondary color pairs. The settings page now lets you apply a preset instantly, while still keeping manual color inputs as the source of truth for saving. Follow-up: preset buttons currently preview and then require Save settings to persist, which is intentional but should stay obvious in the UI.
